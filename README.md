Uber-Dashboard
 Uber NCR Ride Analytics Dashboard built in Power BI using 150K+ ride records. The dashboard analyzes bookings, revenue trends, ride distance, customer &amp; driver ratings, cancellations, CTAT/VTAT metrics, payment methods, and location insights through interactive visualizations, bookmarks, and KPI tracking.

 Uber NCR Ride Analytics Dashboard | Power BI

 Project Overview

This project presents an interactive **Uber NCR Ride Analytics Dashboard** developed in **Power BI** using ride booking data from the National Capital Region (NCR). The dashboard provides comprehensive insights into booking trends, revenue generation, ride performance, customer satisfaction, operational efficiency, and cancellation patterns.

The objective of this project is to transform raw ride-booking data into meaningful business insights that can support strategic decision-making and improve customer experience.

 Key Business Objectives

* Analyze overall ride booking performance.
* Monitor revenue trends and ride demand patterns.
* Evaluate customer and driver ratings.
* Identify high-demand pickup and drop locations.
* Track ride completion and cancellation rates.
* Measure operational efficiency using CTAT and VTAT metrics.
* Understand payment preferences and vehicle utilization.
